Amazon Web Services Pricing Estimate

Compute & AI:

  • Lambda: $0.20/million requests
  • ECS Fargate: $0.04048/vCPU hour
  • Bedrock (Claude): $0.008/1K tokens
  • Kendra: $810/month (Developer)

Storage & Data:

  • DynamoDB: $0.25/million RRU
  • S3 Standard: $0.023/GB/month
  • ElastiCache: $0.017/hour (t3.micro)
  • SQS: $0.40/million requests
Total Estimated Cost: $180-550/month

Based on 100K queries/month with moderate traffic
